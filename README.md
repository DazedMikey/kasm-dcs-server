# DCS Kasm Server

### Clone
```
# ssh
git clone git@github.com:DazedMikey/kasm-dcs-server.git
# https
git clone https://github.com/DazedMikey/kasm-dcs-server.git
```

### Build
```
cd kasm-dcs-server
make build
```
or tag develop with...
```
make build-dev
```

### Push
```
make build push
```

### Release
```
make release
```
